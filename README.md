# NetMind 🧠🔍

Herramienta de red potenciada con IA para análisis en tiempo real.

## Características
- Captura y análisis de tráfico (pcap, live)
- Detección de anomalías con ML
- Clasificación de protocolos
- Diagnóstico automático de problemas
- Multiplataforma (Linux/Windows)

## Instalación
```bash
pip install netmind
